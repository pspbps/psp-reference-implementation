# Local Development (Windows) — PSP Reference Implementation

This guide explains how to compile, test, and locally deploy the PSP v1.0 reference contracts
using **Hardhat** on **Windows**.

> Goal: run everything locally with minimal steps and minimal confusion.

---

## Prerequisites

- Node.js **v20.x LTS** recommended
- Git (optional if you use GitHub ZIP download)
- A terminal:
  - PowerShell, Windows Terminal, or VS Code integrated terminal

---

## Step 1 — Open a Terminal in the Project Folder

1. Open the repository folder on your PC.
2. In the folder window, click the address bar.
3. Type `powershell` and press Enter.
